# 🏦 Sistema Bancário em Python

Aplicação de linha de comando desenvolvida em Python para simulação de operações bancárias, utilizando Programação Orientada a Objetos (POO) e regras de negócio realistas.

---

## 📌 Sobre o projeto

Este projeto simula um sistema bancário completo, permitindo:

* Criação de usuários e contas
* Depósitos e saques
* Consulta de extrato
* Controle de transações
* Aplicação de regras bancárias (limites e validações)

---

## 🧠 Conceitos aplicados

* Programação Orientada a Objetos (POO)
* Herança e encapsulamento
* Classes abstratas (ABC)
* Decorators em Python
* Iteradores personalizados
* Manipulação de datas e horários

---

## ⚙️ Funcionalidades

* 💰 Depósito de valores
* 💸 Saque com limite por operação
* 📄 Extrato com histórico completo de transações
* 👤 Criação de clientes
* 🏦 Criação de contas bancárias
* 🔒 Controle de:

  * Limite de saques
  * Limite de transações diárias
  * Validação de saldo

---

## 🏗️ Estrutura do sistema

* `Cliente` → gerenciamento de usuários
* `Conta` → operações bancárias
* `ContaCorrente` → regras específicas (limites)
* `Historico` → armazenamento de transações
* `Transacao` → classe abstrata para operações
* `Deposito` / `Saque` → implementação das transações
